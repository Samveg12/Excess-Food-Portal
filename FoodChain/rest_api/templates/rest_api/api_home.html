Rest API

If you want to extract information about the Food Available, Food Requests made or other details, then you are at the right place
Our REST API handles requests and gives responses in JSON format.

base_url = "localhost:8000/API/"
To get information about available food: base_url + "available_food/"
To get information about Requests made for food: base_url + "food_request/"
To get Details about Users: base_url + "user_details/"
